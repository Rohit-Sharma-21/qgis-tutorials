使用 QGIS 瀏覽器¶

QGIS 具有一個稱為 QGIS 瀏覽器 （QGIS Browser）的獨立應用程式，是 QGIS 的實用附屬工具，用來管理檔案也相當方便。如果你也是 ArcGIS 使用者，你就會知道它的功用其實就與 ArcCatalog 非常類似。

尋找 QGIS 瀏覽器¶

QGIS 瀏覽器獨立應用程式¶

QGIS 瀏覽器（QGIS Browser）是 QGIS 標準安裝的一部份：

  • Windows：如果你是從 OSGEO4W 安裝檔安裝 QGIS 的話，可以在開始 -> 所有程式中找到 QGIS Browser。

  • Mac: 應用程式位於 QGIS.app/Contents/MacOS/bin/QGIS Browser.app，你也可以建立一個捷徑：前往應用程式所在資料夾，在 QGIS 圖示上按右鍵，選擇 Show Package Contents，前往 MacOS ‣ bin ‣ QGIS Browser，在 QGIS Browser 圖示上按右鍵，選擇 Make Alias，然後把 QGIS Browser alias 拖到 應用程式 資料夾，現在你就可以用和其他應用程式一樣的方法開啟 QGIS Browser。

  • Linux：可以使用 qbrowser 這個指令執行 QGIS 瀏覽器。應用程式的位置與 QGIS 主程式的資料夾位置相同。

嵌在 QGIS 中的瀏覽器面板¶

另一個輕鬆使用 QGIS 瀏覽器的方法是透過 QGIS 主程式本身來執行。瀏覽器面板位於 QGIS 左手邊面板 (底部)，點選 Browser 分頁，就可開啟 QGIS 瀏覽器。如果你沒有看到 Browser 分頁，要前往 檢視 ‣ 版面配置 ‣ Browser 以開啟瀏覽器。(舊版 Linux 則是位於 設定 頁面而不是 檢視 頁面。)

操作流程¶

  1. 我們現在來看看 QGIS 瀏覽器的一些功能。開啟 QGIS 瀏覽器的獨立應用程式，前往你在系統中存放 GIS 資料的資料夾，你就會發現使用此瀏覽器的優點──只有 QGIS 支援的空間圖層檔案會顯示出來，不是地理空間資料的檔案室不會顯示的。請點選任一圖層。

  1. 選擇圖層之後，詮釋資料會在右側面板的第一個分頁中顯示。你可以快速瀏覽資料的基本資訊，像是圖徵數量、投影方式等等。

  1. 切換到「預覽」分頁就可以預覽資料。這可以讓你在使用 QGIS 開啟資料前快速地查閱它們看起來的樣子。

  1. 最後一個分頁是「屬性」分頁，這可以讓你查閱圖層的屬性表格，包括所有的欄位和屬性值。

  1. QGIS 瀏覽器不只可以查閱檔案系統中的向量與影像圖層，更可以直接存取資料庫和網路資源。如果你有藉由 WMS 使用任何線上資料的話，在此瀏覽器中也可以快速的預覽它們：打開 WMS 的資料夾選單，就可以看到之前已經設定好的線上資源。如果你有 PostGIS、SpatialLite 或 MSSQL 資料庫的話，也可以用類似的方法在瀏覽器中存取。

  1. QGIS 瀏覽器也可以直接查看和開啟 .zip 檔案。前往有 .zip 檔的資料夾後，就可以看到 .zip 檔會像是其他支援的檔案類型一樣出現在列表，瀏覽的方法就像是其他種類的資料一樣。

  1. 另一個有用的功能是標記系統中的某個資料夾為 我的最愛，只需右鍵點選任何資料夾，然後選擇 增加為喜歡。

備註

把資料夾加入最愛列表的功能目前只能在 QGIS 中的瀏覽器面板操作，獨立的 QGIS 瀏覽器並不支援這項功能。

  1. 加到喜歡列表後，就可以直接從 喜歡 資料夾進入此資料夾了。

  1. 當你選擇某圖層後，按兩下或是拖曳圖層到畫布上就可直接加入 QGIS 畫布。

  1. 現在你可以把透過左下方的按鈕把左邊的面板切到 圖層 分頁，以在 QGIS 中查看剛加入的圖層。
